# Resume Keyword Scanner – Chrome Extension

A Chrome extension that analyzes job descriptions against your resume to identify missing keywords and improve ATS alignment.

## Features
- Paste or upload resume text
- Upload PDF resumes (text-based or scanned)
- OCR support for image-based PDFs
- Paste job descriptions from any site
- Keyword match scoring and missing keyword highlights
- Local-only processing (no data leaves the browser)

## Why This Exists
Many resumes are rejected by ATS systems due to missing keywords. This tool helps job seekers quickly compare their resume to real job postings and improve match rates.

## Privacy
- No data collection
- No tracking or analytics
- No external servers
- All processing happens locally in the browser

## Tech Stack
- JavaScript (Chrome Extensions API, Manifest V3)
- PDF.js for text extraction
- Client-side OCR
- HTML / CSS for UI

## Installation (Developer Mode)
1. Clone this repository
2. Open Chrome → `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder
